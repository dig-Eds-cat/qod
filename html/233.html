Search RIB

This on-line edition is fully indexed and searchable. Here's some guidance:

  • If you already know the number of the inscription from RIB, or tablet from the Vindolanda or Bloomberg Tablets, you could Search by number. Note that only those inscriptions in RIB volumes I (1-2400) and III (3001-3550) may currently be found.
  • Enter the search term you are interested in, e.g., Mithras, Vindolanda, or Caracalla.
  • Searching is case-insensitive. This means searching for IMPERATOR will return exactly the same results as imperator.
  • Boolean searches are supported. If you want to see inscriptions that mention Mars or Cocidius, enter Mars OR Cocidius. Likewise, if you want to see inscriptions that mention Mars and Cocidius, enter Mars AND Cocidius. You must use capital letters for OR or AND - otherwise the search engine will interpret or (or and) as a search term.
  • Wild card searching is supported. E.g., searching for Sul* will return results including Suleviae, Sulinus, Sulis, Sulpicius, etc.
  • Searches for phrases consisting of more than one word should be enclosed within double quotation marks. E.g., searching for "Old Carlisle" will return results containing the phrase "Old Carlisle", but omit inscriptions containing only Old and only Carlisle.
  • After you receive your first set of search results, you will have the option to apply further filtering criteria.
  • Note that all Latin words have been lemmatised. This means searching for 'cohors' will return inscriptions containing cohors, cohortis, etc.
  • The search functionality currently searches the texts of each inscription, as well as its English translation and all associated metadata. It is hoped that the filtering function can help to reduce the number of search results. An advanced search interface is being developed that will permit more fine-grained search queries and give the user significantly more control over search results.
  • Dates: Given the inherent difficulty in dating inscriptions, most have not been assigned a particular date or range of dates any more specific than those of the traditionally recognized Romano-British era (a.d. 43-410). This means that queries for a certain range of dates will be over-inclusive, since all undated inscriptions will necessarily fall within any range specified. To avoid this, a checkbox labeled "Included undated inscriptions" has been added to the Date Range filter panel, which is not ticked by default. If you wish search for all inscriptions that may possibly fall within the date range specified, dated or undated, simply tick this box.

Vindolanda Tablets
View the inscription

The Vindolanda Tablets from the late 1st and early 2nd centuries are among the most important Roman artifacts ever excavated in Britain. This set of nearly 780 texts written on thin sheets of wood comprises a rich source for military, economic and social history, and language.

Bloomberg Tablets
View the inscription

The Bloomberg wooden writing tablets were found in waterlogged deposits during excavations for the Bloomberg European HQ in the City of London. The over 400 tablets yielded around 80 texts giving insights into the first fifty years following the Roman conquest of Britain.
